# 최빈값

## 문제

최빈값은 주어진 값 중에서 가장 자주 나오는 값을 의미합니다. 

정수 배열 `array`가 매개변수로 주어질 때, 최빈값을 return 하도록 solution 함수를 완성해보세요. 

최빈값이 여러 개면 -1을 return 합니다.



## 오답

```python
def solution(array):
    max_cnt = 0
    for i in array:
        if array.count(i) > max_cnt :
            max_cnt = array.count(i)
            answer = i
    
    return answer
```

        count = 0
